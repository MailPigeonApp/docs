## Improving the docs

Documentation is extremely important and takes a fair deal of time and effort to write and keep updated. Everything is written in Markdown to facilitate the process of contributing.

<Accordion title="Docs Setup Guide" icon="github">
    ### Docs Setup Guide
    1. Clone the repository
    
        ```bash
        git clone https://github.com/MailPigeonApp/docs.git
        ```

    2. Install the dependencies

        ```bash
        yarn install
        ```

    3. Start the development server

        ```bash
        yarn dev
        ```

    4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

</Accordion>

## Opening Issues
